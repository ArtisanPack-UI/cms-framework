# Core Features

The ArtisanPack UI CMS Framework provides a comprehensive set of features to build powerful content management systems. This section covers all the core functionality available in the framework.

## Feature Categories

### [Administration](core-features/administration)
Complete administrative interface and dashboard management:
- **[Admin Menus](core-features/administration/admin-menus)** - Register and manage administrative pages and menus
- **[Dashboard Widgets](core-features/administration/dashboard-widgets)** - Create and manage customizable dashboard widgets
- **[Implementing Dashboard Widgets](core-features/administration/implementing-dashboard-widgets)** - Detailed widget implementation examples

### [Security](core-features/security)
User management, authentication, and security features:
- **[Users and Roles](core-features/security/users)** - User management and permissions system
- **[API Authentication](core-features/security/api-authentication)** - API authentication using Laravel Sanctum
- **[Sanctum Setup](core-features/security/sanctum-setup)** - Detailed Sanctum configuration guide
- **[Audit Logging](core-features/security/audit-logging)** - Authentication events and user activity logging
- **[Two-Factor Authentication](core-features/security/two-factor-authentication)** - Enhanced security with 2FA
- **[Two-Factor Setup](core-features/security/two-factor-setup)** - Step-by-step 2FA implementation

### [Content Management](core-features/content)
Content creation, management, and organization:
- **[Content Types](core-features/content/content-types)** - Define and manage different types of content
- **[Settings](core-features/content/settings)** - Application settings management
- **[Notifications](core-features/content/notifications)** - User notifications and messaging system
- **[Notification Implementation](core-features/content/notification-implementation)** - Implementing notification support

### [Progressive Web App (PWA)](core-features/pwa)
Modern web application features:
- **[PWA Overview](core-features/pwa)** - Progressive Web App functionality overview
- **[PWA Integration Guide](core-features/pwa/pwa-integration-guide)** - Detailed PWA implementation guide
